# scss-replace


这是一个可以简单生成和替换scss变量的脚本.


## Usage
  **Step 1:** npx:
  - npx scss-replace *.config.js

## Usage
  **Step 2:** Install plugin:
  - npm install scss-replace



## example

- 配置文件请参考 example
- demo
- npx scss-replace ./example/replace.config.js
